# Reflex Showcase App

An interactive demonstration app showcasing custom Reflex components and utilities for building Constellab applications. Each component includes a live demo, source code, and documentation.

## Components

### Resource Access
- **Get Resources** - Access input resources passed to the app from Constellab
- **Get Parameters** - Retrieve configuration parameters from the app settings

### Authentication
- **Get Current User** - Access information about the currently authenticated user

### Rich Text
- **Rich Text Editor** - Interactive editor supporting formatting, headings, lists, formulas, and more

### User Components
- **User Profile Picture** - Display user avatar or initials in a circular avatar (normal and small sizes)
- **User Inline Component** - Show user avatar with full name in horizontal layout
- **User Select** - Dropdown component for selecting users from a list

### Documentation
- **Doc Component** - Automatically generate documentation for functions and components from docstrings
